# 🎮 Rock Paper Scissors Web App

A modern, interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. Features a beautiful gradient design, smooth animations, and engaging gameplay.

🌐 **Live Demo:** [Play the Game](https://rock-paper-scisssor1.netlify.app/)

## ✨ Features

- **Interactive Gameplay**: Play against the computer with intuitive controls
- **Score Tracking**: Keep track of wins for both player and computer
- **Round Counter**: 20-round matches with clear progress indication
- **Auto Play Mode**: Watch the computer play against itself
- **Responsive Design**: Works beautifully on desktop and mobile devices
- **Visual Feedback**: Color-coded messages and animations
- **Victory Celebration**: Fireworks animation when you win
- **Modern UI**: Gradient backgrounds and sleek design elements

## 🎯 How to Play

1. Choose between Rock, Paper, or Scissors by clicking on the corresponding icon
2. The computer will randomly select its choice
3. See who wins the round based on classic rules:
   - ✊ Rock beats ✌️ Scissors
   - ✌️ Scissors beats ✋ Paper
   - ✋ Paper beats ✊ Rock
4. The first to win the majority of 20 rounds wins the game!
5. Use the "Auto Play" button to watch the computer play itself
6. Reset the game at any time with the "Reset Game" button

## 🛠️ Tech Stack

- **HTML5** - Semantic structure and game layout
- **CSS3** - Modern styling with gradients, flexbox, and animations
- **JavaScript** - Game logic and interactive functionality
